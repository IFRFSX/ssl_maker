1. 这是一个调用GNU/Linux操作系统的OpenSSL程序的小程序，
2. 它可以帮你在极短的时间内生成超多的SSL证书！！！！
3. 没错，按照提示，输入好你需要的数量和证书位宽，然后按住Enter！！！！！！！！
4. 最好，用一个小重物压住Enter键！！！！！！！！
5. 最后，成千上万的SSL证书就会生成到命令行所指目录的“SSL文件”文件夹！！
